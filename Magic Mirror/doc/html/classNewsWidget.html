Detailed Description

This class handles the initialization of APIRequest class.

This class is used to initialize and configure all the settings in order to use the APIRequest class. This will pull the headlines, the pictures, descriptions and the links to articles.

This function changes the news story that is being shown to the user.

This function is responsible for updating the story that is being displayed to the user after a certain amount of time. It will cycle the array that contains the stories information forever so that the user has a chance to see a story again if they miss it the first time.
