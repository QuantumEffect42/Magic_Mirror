◆ parse_by_line()

mime& mailio::mime::parse_by_line ( const std::string &  line,
bool  dot_escape = false 
)

Parsing a line of mime part.

The method is used to parse message string line by line, as received over the protocols. For each line a CRLF is appended. If the line is CRLF only, then the parsing ends.

If a message with mime parts is parsed but the message also contains a content, then the content is stored instead to be skipped.

The method relies on the virtual parse_header_line() which should specify headers of interest to the class. The message class uses it for its own headers.

Parameters
lineString to be parsed. If it's CRLF, then message parsing ends; any further parsing is undefined.
dot_escapeFlag if the leading dot should be escaped.
Returns
Mime itself.
Exceptions
*parse_header(), parse_content.
